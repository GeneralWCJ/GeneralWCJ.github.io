Unnamed Full Stack Project

This project started off with my mom giving me an idea. She is a teacher and found a resource of problems that built upon each other. She wanted my help to try to create a sort of game to entice people to try to finish the fiendishly hard problems that this website had.

Then mom decided that she didn't want to pay, but the project still seemed fun, and it would still be good on a resume, so I decided to continue on but genericicing the plan to make it usefull for more things than what she wanted.
